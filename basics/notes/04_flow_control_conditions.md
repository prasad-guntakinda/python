## Practices:
1. Even or Odd: Take a number and print whether it is even or odd.
2. Largest of 3 Numbers: Take 3 numbers and print the largest using if-elif.
3. Grade Calculator
    ````text
    Take marks as input and print grade:
    ≥90 → A
    ≥75 → B
    ≥50 → C
    else → Fail
    ````
4. Leap Year Checker: Check whether a given year is leap year or not.
5. Electricity Bill Calculator

````text
Input units consumed:
    First 100 units → ₹5/unit
    Next 100 → ₹7/unit
    Above 200 → ₹10/unit
Calculate total bill.
````
6. Login System

    ````text
    Given:
    username = "admin"
    password = "1234"

    Take input and validate login.
    ````
7. Triangle Type

````text
    Given 3 sides, print whether triangle is:

    Equilateral
````
